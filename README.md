## How to use


## Local usage

1. Clone the project to your local computer for exampy by using:
```bash
git clone https://github.com/schmelto/wordle_solver.git
```

2. Go into the cloned folder (`cd wordle_solver`) and install the dependencies by running:
```bash
npm install
```
